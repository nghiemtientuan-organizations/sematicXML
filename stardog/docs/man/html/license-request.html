stardog-admin license request Manual Page

NAME


stardog-admin license request —(Deprecated) Request a Stardog license.

SYNOPSIS


stardog-admin license request [ --force ] [ --output <output> ] [ --quiet ] [ --stardog-home <STARDOG_HOME> ] [ {-v | --verbose} ]

OPTIONS


--force
Overwrite the license file if it exists.
--output <output>
A path to the location where the license will be written.
--quiet
Minimize the output.
--stardog-home <STARDOG_HOME>
The stardog home for this license.
-v, --verbose
Flag that can cause more detailed information to be printed such as errors and status. Exact output depends upon the command and options used.

DISCUSSION


This command has been deprecated. Please visit https://stardog.com to acquire a new license.
